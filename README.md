# Legacy RPG
Legacy RPG *will* be a massive open-source rpg game built upon a discord bot.


### Command
##### Admin
* !prefix <prefix>

##### Info
* !botinfo
* !help
* !patch

* !character <create, delete>

* !inventory
* !collect <wood, minerals, fish, prey>
* !craft <item>
* !smelt <refuel, item>
* !stats
* !combat
* !coins
